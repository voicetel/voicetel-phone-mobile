# VoiceTel Phone

A cross-platform WebRTC SIP Phone built with Electron.

## Features

- WebRTC-based SIP calling
- Cross-platform support (macOS, Windows, Linux)
- Microphone and media permissions
- Clean, focused interface
- Auto-hiding menu bar

## Quick Start

### Prerequisites

- Node.js 16+ and npm
- A high-quality icon file (1024x1024 PNG) saved as `icon-source.png`

### Setup

1. **Clone or download the repository**

2. **Run the setup script** (creates icon files and directory structure):
```bash
   chmod +x setup-project.sh
   ./setup-project.sh
